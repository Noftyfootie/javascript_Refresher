# JavaScript Refresh and Modern ES6 Concepts

A minimalist, interactive web sandbox built with HTML5 and Tailwind CSS to practice and visualize core JavaScript and ES6 fundamentals. The webpage provides a sleek user interface, while the underlying logic outputs structured results directly to your browser's developer tools.

## Learning Outcomes Demonstrated

1. **Presentation Through Console**: Demonstrates backend array manipulation using advanced modern array methods (`.map()`, `.forEach()`) and showcases structured debugging using clean `console.group()` folders.
2. **Implementation on an HTML Webpage**: Demonstrates active DOM manipulation by transforming raw JavaScript arrays into functional, dynamically injected HTML UI components without hardcoded elements.

## Features

- **Clean UI Layout**: Styled with Tailwind CSS, utilizing a fully centered Flexbox structure, responsive typography, and smooth utility classes.
- **Structured Console Logging**: Uses `console.group()` and `console.groupEnd()` to organize JavaScript logs into neat, expandable sections.
- **Comprehensive JS Refresh**: Covered concepts include:
  - **Scope**: Differences between `var`, `let`, and `const`.
  - **Arrow Functions**: Modern syntax paired with ES6 Template Literals.
  - **Object Destructuring**: Extracting properties efficiently.
  - **Array Methods**: Practical examples of `.forEach()`, `.map()`, `.filter()`, and `.reduce()`.

## Project Structure

## How to Run the Project

1. Clone or download this project repository to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Edge, Firefox, Safari).
3. Open your browser's **Developer Tools**:
   - Right-click anywhere on the page and select **Inspect**.
   - Or press `F12` (`Cmd + Option + I` on Mac).
4. Click on the **Console** tab to view the categorized JavaScript code output.
